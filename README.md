# ATR Calculator

A web-based trading tool for calculating **Average True Range (ATR)** and **True Range (TR)** to measure market volatility.

## 🔗 Live Demo
https://sherryx-dev.github.io/atr-calculator/

---

## 📌 Overview
This project provides a simple and practical ATR calculator for traders who want to measure volatility and improve risk management decisions.

ATR does not predict direction — it measures volatility.

---

## ⚙️ Features
- Calculate True Range (TR)
- Calculate Average True Range (ATR)
- Clean and responsive UI
- Built using pure HTML, CSS, and JavaScript

---

## 🧠 How ATR Works
True Range (TR) is calculated as the maximum of:
- High - Low
- |High - Previous Close|
- |Low - Previous Close|

ATR is typically calculated using a 14-period average.
